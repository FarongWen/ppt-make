# 党建自动化脚本使用说明

本项目包含多个用于党建工作自动化的Python脚本，主要功能包括Excel数据处理、PPT生成和党员信息格式化等。

## 快速开始

### 第一次使用前准备
1. **安装Python环境**（Python 3.6+）
2. **安装依赖包**：
   ```bash
   pip install pandas python-pptx openpyxl
   ```
3. **准备必需文件**：
   - PPT模板文件：`ppt_template.pptx`
   - 党员数据Excel文件
   - 模板配置文件：`requirement.md`

### 最常用的操作

**生成党员转正PPT：**
```bash
# 1. 处理转正数据
python trans.py

# 2. 生成PPT（每页5行）
python process_ppt.py --excel 党员转正信息.xlsx --rows 5 --output 转正信息.pptx
```

**生成新党员发展PPT：**
```bash
# 1. 处理发展数据
python develop.py

# 2. 生成PPT
python process_ppt.py --excel party_members_info.xlsx --rows 4 --output 发展信息.pptx
```

## 脚本详细说明

### 1. process_ppt.py - PPT生成脚本

**功能描述：**
- 读取Excel文件数据，将每行数据拼接后填充到PPT模板中
- 支持自定义每页显示的行数（4或5行）
- 自动复制页面，直到所有Excel数据都填充完成

**使用方法：**
```bash
python process_ppt.py [选项]
```

**命令行参数：**
- `--excel EXCEL_FILE`: 指定Excel文件路径（默认：data.xlsx）
- `--template TEMPLATE`: 指定PowerPoint模板文件路径（默认：ppt_template.pptx）
- `--output OUTPUT`: 指定输出文件路径（默认：output.pptx）
- `--rows {4,5}`: 每页显示的行数，必须是4或5（默认：4）

**使用示例：**
```bash
# 使用默认参数
python process_ppt.py

# 自定义参数
python process_ppt.py --excel data.xlsx --rows 5 --output custom_output.pptx
```

**文件要求：**
- 模板PPT中需要包含一个包含"陈芝琳"文本的文本框作为内容填充目标
- Excel文件格式标准，支持多列数据

## 工作流程建议

### 处理新党员发展信息：
1. 准备`2025年秋季学期新发展党员信息.xlsx`文件
2. 运行`python examine_excel.py`检查数据格式
3. 运行`python develop.py`生成格式化信息
4. 使用生成的`party_members_info.xlsx`作为数据源
5. 运行`python process_ppt.py --excel party_members_info.xlsx`生成PPT

### 处理党员转正信息：
1. 准备`转正.xlsx`文件
2. 运行`python examine_xlsx.py`检查数据格式
3. 运行`python trans.py`生成格式化信息
4. 使用生成的`党员转正信息.xlsx`作为数据源
5. 运行`python process_ppt.py --excel 党员转正信息.xlsx`生成PPT

## 日期格式处理

所有脚本都包含智能日期格式处理功能，支持：
- Excel时间戳格式自动转换
- 字符串日期格式解析
- 统一输出为中文日期格式（如：2025年12月02日）
- 处理缺失日期值（显示为"未知日期"）

## 注意事项

1. **文件路径**：确保所有输入文件存在于脚本运行目录中
2. **Excel格式**：确保Excel文件列名与脚本中的字段映射一致
3. **模板文件**：PPT模板需要包含特定的文本框标识
4. **编码问题**：所有文件使用UTF-8编码
5. **权限问题**：确保脚本有读写权限

## 故障排除

### 常见错误及解决方案：

1. **文件未找到错误**
   - 检查文件名和路径是否正确
   - 确保文件存在于当前目录

2. **模板未找到错误**
   - 检查PPT模板是否包含标识文本
   - 确保requirement.md包含正确的模板格式

3. **列名不匹配错误**
   - 使用检查脚本验证Excel文件结构
   - 根据实际列名调整脚本中的字段映射

4. **日期格式错误**
   - 脚本已包含多种日期格式的处理逻辑
   - 如遇特殊格式，可在format_date函数中添加处理逻辑
